# Anime-Verse
=== Project akhir lab mobile ===
Nama: Jason Sanjaya 
NIM: 231401066

Anime-Verse adalah aplikasi mobile berbasis Flutter yang dibuat sebagai proyek akhir pada mata kuliah Lab Mobile 5.
Aplikasi ini menampilkan beberapa tampilan pengguna (UI screens) yang menggambarkan alur dasar aplikasi mobile modern, seperti login, signup, home, dan fitur lainnya.
Proyek ini bertujuan untuk:
- Memahami struktur aplikasi Flutter
- Mengimplementasikan navigasi menggunakan GoRouter
- Mendesain UI responsif
- Menampilkan alur autentikasi sederhana (login & signup)
- Membangun aplikasi hingga dapat dijalankan pada Android
